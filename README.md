# Discord Bot JediStar

## Usage

> !raid
Affiche l'aide

> !raid rancor P1 65.6% 42,3
Estime les dégâts effectués en Phase 1 du raid rancor.
A noter : le symbole % est optionnel, virgule et point pour le séparateur décimal interchangeables.
